## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/slider-weight.git
```

Go to the project directory

```bash
  cd slider-weight
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
